# Pixiv_Wallpaper_for_Win10
抓取Pixiv.net的图片并设置为Win10壁纸的UWP APP，需要在Windows10 1709以上的版本环境中运行。
有top50与"为你推荐"两种模式，"为你推荐"需要登录账号。
目前登录操作(为你推荐模块)在中国大陆区域需要走代理才能连接，Top50功能不受影响可以直接连接
